# AGENTS.md

The canonical agent guidelines for this repository live in
**[`.agent/AGENTS.md`](.agent/AGENTS.md)** — read that file before making changes.

It covers the FeatureOps rules every assistant must follow here: flag-gating risky changes
(especially payments), the flag naming convention, explicit project scoping for the Unleash MCP
server, and the instance-level vs. project-level entity model.
